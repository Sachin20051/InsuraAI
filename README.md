# InsuraAI
AI-powered analytics dashboard that lets users upload datasets and query them using natural language, auto-generating SQL, charts, and KPI metrics in real time.
An intelligent full-stack analytics dashboard built with React, FastAPI, and Gemini AI. Users can upload CSV datasets and ask questions in plain English — the AI converts them into SQL queries, executes them against a SQLite database, and renders dynamic bar, line, and pie charts alongside KPI metrics. No SQL knowledge required.
Features: Natural language querying · AI-generated SQL · Dynamic chart rendering · CSV file upload · KPI metrics · Follow-up conversational queries · Dark UI with responsive design
